Preprocessing Steps — PySpark
  1. 01Null value imputation using column medians
  2. 02Categorical encoding (location, sensor, weather)
  3. 03Min-max normalisation → [0, 1]
  4. 04Temporal feature extraction (hour, day, season)
  5. 05VectorAssembler → feature vector
  6. 0680/20 stratified train-test split
  7. 07Gaussian noise augmentation (synthetic expansion)
